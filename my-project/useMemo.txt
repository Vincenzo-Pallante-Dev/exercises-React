Create a `FilteredList` component that receives a list of objects each containing a `name`, an `id` and a `age` prop. The `FilteredList` component should render only the items of the list whose `age` is greater than 18, and the filtering should only happen when the list changes. Use `useMemo` to memoize the filtered list.
Crea un componente "FilteredList" che riceve un elenco di oggetti,
 ciascuno contenente un "nome", un "id" e una prop "età". 
 Il componente "FilteredList" 
 dovrebbe eseguire il rendering solo degli elementi dell'elenco la cui "età" è maggiore di 18
  e il filtraggio dovrebbe avvenire solo quando l'elenco cambia. Utilizza `useMemo` per memorizzare l'elenco filtrato.
